NeuroTrack

Still in development, coming soon!

Tags: Swift, Model View Controller pattern, IOS programming, object oriented programming and data analysis

NeuroTrack is an IOS app I developed in collaberation with Darrion Shack. This app uses the IOS Core Motion, Core Data, and Swift Graph APIs to track, analyze, and model the tremors and gait patterns of patients with Parkinson's disease to see how their symptoms are changing over time.

Technical Details

This program is implemented using the Model View Controller design pattern. The view is the interface the user interacts with, the controller sends user input to the model, and the model serves as central database for all the data collected. The model also analyzes the data by finding the standard deviation of the data over a set period of time to understand the progression of their symptoms. The patient data is collected by using Core Motion's APIs to access the IPhone's pedometer, gyroscope, and accelerometer. Persistant storage is handled using the Core Data API to save user data and Swift Graph is used to graph the data after it has been processed in the model.
